Void Scribe

About:

Void Scribe is a publicly available API that currently supports basic generation of names through Markov Generation as well as creation of basic myth, quest, adjective, hero, and tale structures. Void Scribe will be developed into a publicly available Narrative Generation API. Void Scribe is being developed as part of a school project by team MachineLearningMadlads.
